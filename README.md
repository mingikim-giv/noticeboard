# Map 활용 콘솔 게시판 만들기

## 게시판 설명
* 어느 유저가 게시판을 이용하기 위해 찾아왔다. 게시판을 이용하려고 했는데 회원가입을 해야되네..? 유저가 회원가입을 하고 로그인을 했더니 게시판에 글 쓰기/수정/삭제 게시판 보기 기능이 활성화 되었다. 게시판에 이제 볼일이 없는 유저는 회원 탈퇴를 통해 더 이상 게시판을 이용 하지 않기로 했다.

### 게시판 기능
1. 회원가입
2. 회원탈퇴
3. 로그인
4. 로그아웃
5. 글 메뉴
    * 글 쓰기
    * 글 수정
    * 글 삭제
    * 게시글 보기 
6. 종료
